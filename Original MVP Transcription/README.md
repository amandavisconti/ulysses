Ulysses
=======
This directory contains a reliable digital text of James Joyce's novel *Ulysses*: the plaintext transcription of the 1922 first printing of Ulysses by **Matthew Kochis** and **Patrick Belk** with the **[Modernist Versions Project] (http://web.uvic.ca/~mvp1922/)**.

This file is **licensed [CC BY SA](https://creativecommons.org/licenses/by-nc-sa/3.0/)** by the Modernist Versions Project text and is placed here for preservation (Lots of Copies Keeps Stuff Safe) and comparison (with the corrected and encoded files also available in this repository) purposes. You can read more about the Modernist Versions Project at [http://web.uvic.ca/~mvp1922](http://web.uvic.ca/~mvp1922/).
